task1.js
Данная функция принимает обьект и с помощью for in пробегает по свойствам объекта.
С помощью метода hasOwnProperty она выводит в консоль все собственные ключи и значения объекта

task2.js
Данная функция принимает строку и объект, после с помощью for in пробегает по объекту.
Во время пробега по объекту происходит сравнение имени свойства со строкой, и если оно сходится,
то выводит в консоль true иначе false

task3.js
Данная функция при помощи Object.create(null) создает и возвращает пустой объект без прототипа

task4.js
Создано консольное приложение электрических приборов, где вводится их название, иерархия.
В ней есть функции вычисление потребления мощности приборов, которая принимает киловаты и время работы прибора.
Так же есть функция подклчения к розетке(true\false).
Индивидульные функции каждого прибора:
 - Функция наличия лампочки в лампе;
 - Функция переключения каналов в телефизоре.
Есть делегирующая связь, которая принимает имя и тип прибора.
